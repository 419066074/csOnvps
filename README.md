# cobaltstrike on server
cobaltstrike4.4 自动化执行teamserver，快速临时开启Teamserver脚本。
Ubuntu VPS

- 密码为随机uuid->md5sum 值
- teamserver修改变量改为：随机生成key、password、port等
```bash
curl cs.0xc2.cn/run | bash
```
